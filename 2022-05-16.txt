 1) Scanner 변수 생성 규칙 : Scanner 변수 = new Scanner(System.in);
 2) 변수.next()(nextInt(), next*()...)하여 사용자가 입력한 값을 변수에 대입
------------------------------------------------------------------------------------------
package day02;

import java.util.Scanner;

public class quiz01 {
	public static void main(String[] args) {
		Scanner input = new Scanner(System.in);
		String name = "";
		String resultName = "";
		int korean = 0;
		int english = 0;
		int meth = 0;
		int sum = 0;
		
		System.out.print("당신의 이름은 무엇입니까?");
		name = input.next();
		
		for (int i=0; i<name.length(); i++) {
			resultName += name.charAt(i) ;
			if (i < name.length()-1) {
				resultName += " ";
			}
		}
		
		System.out.print(name + "님의 국어 점수 : ");
		korean = input.nextInt();
		System.out.print(name + "님의 영어 점수 : ");
		english = input.nextInt();
		System.out.print(name + "님의 수학 점수 : ");
		meth = input.nextInt();
		sum = korean + english + meth;
		
		System.out.println("===============");
		System.out.println("이 름 : " +  resultName);
		System.out.println("===============");
		System.out.println("국 어 : " +  korean);
		System.out.println("영 어 : " +  english);
		System.out.println("수 학 : " +  meth);
		System.out.println("===============");
		System.out.println("합 계 : " +  sum);
		System.out.println("===============");
	}
}
------------------------------------------------------------------------------------------
당신의 이름은 무엇입니까?장혜인
장혜인님의 국어 점수 : 1
장혜인님의 영어 점수 : 2
장혜인님의 수학 점수 : 3
===============
이 름 : 장 혜 인
===============
국 어 : 1
영 어 : 2
수 학 : 3
===============
합 계 : 6

8. 논리 연산자 : 참 또는 거짓 결과
 1) &&(and) : 모두가 참일때 참
 2) ||(or) : 하나라도 참이면 참
 3) !(not) : 반전시켜줌
------------------------------------------------------------------------------------------
		// 논리 연산자 : 참 또는 거짓 결과
		/*
		 * &&(and) : 모두가 참일때 참
		 * ||(or) : 하나라도 참이면 참
		 * !(not) : 반전시켜줌
		 */
		System.out.println("[ 논리 연산자 ]");
		System.out.println("------------------");
		
		int su3;
		su1 = 10; su2 =20; su3 =30;
		System.out.println("===== and =====");
		System.out.println(su1>su2 && su1>su3);
		System.out.println(true && true);
		System.out.println(true && false);
		System.out.println(false && true);
		System.out.println(false && false);
		
		System.out.println("===== or =====");
		System.out.println(su1>su2 || su1>su3);
		System.out.println(true || true);
		System.out.println(true || false);
		System.out.println(false || true);
		System.out.println(false || false);
		
		System.out.println("===== not =====");
		System.out.println( !true );
		System.out.println( !false );

9. 증감 연산자 : 결과적으로 자기자신을 1증가 또는 감소
 1) 전치 또는 전위(++a, --a) : 먼저 자신을 증감시킨 뒤 연산
 2) 후치 또는 후위(a++, a--) : 모든 연산이 끝난 뒤 자기 자신을 증감
------------------------------------------------------------------------------------------
		// 증감 연산자 : 결과적으로 자기 자신을 1증가 또는 감소
		/*
		 a =10;
		 ++a : 전치 또는 전위
		 a++ : 후치 또는 후위
		 a++, ++a => a = a + 1;
		 a--, --a => a = a - 1;
		 */
		System.out.println("[ 증감 연산자 ]");
		System.out.println("------------------");
		
		su1 = 10;
		su1++;
		System.out.println(su1);
		
		su1 = 10;
		++su1;
		System.out.println(su1);
		
		su1 = 10;
		su2 = su1++; // 모든 연산이 끝난 뒤 증가 -> 10 + 1
		System.out.println("su1 : " + su1);
		System.out.println("su2 : " + su2);
		
		su1 = 10;
		su2 = ++su1; // 먼저 자신을 증가시킨 뒤 연산
		System.out.println("su1 : " + su1);
		System.out.println("su2 : " + su2);
------------------------------------------------------------------------------------------
[ 증감 연산자 ]
------------------
11
11
su1 : 11
su2 : 10
su1 : 11
su2 : 11
------------------------------------------------------------------------------------------


10. 삼항 연산자(조건 연산자)
------------------------------------------------------------------------------------------
		// 삼항 연산자(조건 연산자)
		/*
		 변수 = 식 ? 참(값) : 거짓(값);
		 */
		System.out.println("[ 삼항(조건) 연산자 ]");
		System.out.println("------------------");
		
		su1 = 8;
		String s = (su1 % 2 == 0)? "짝수" : "홀수";
		System.out.println(su1 + " => "+ s);
		
		su1 = 11;
		s = (su1 % 2 == 0)? "짝수" : "홀수";
		System.out.println(su1 + " => "+ s);
------------------------------------------------------------------------------------------

------------------

11 => 홀수
